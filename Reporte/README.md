# Reporte

Contiene el artículo científico del proyecto en formato IEEE para conferencias.

## Archivos esperados
- `Reporte_IEEE_Biofeedback_ECG_EEG.pdf` — Artículo final en PDF (máx. 4 páginas)
- `Reporte_IEEE_Biofeedback_ECG_EEG.docx` — Archivo editable Word (plantilla IEEE)

## Instrucciones
1. Usar la plantilla oficial de conferencias IEEE
2. Extensión máxima: **4 páginas**
3. Idioma: **Español**
4. Incluir:
   - Clasificación de las señales (ECG derivación II, EEG frontal/interhemisférico)
   - Descripción del procesamiento (filtros, FFT, detección R, HRV)
   - Justificación matemática: ecuaciones de FFT, RMSSD, LF/HF
   - Pruebas con al menos 2 sujetos experimentales con capturas de la GUI

## Nota
No se admiten reportes en formato LaTeX. Solo .docx y .pdf.
